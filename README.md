# BTC Polymarket Trading Bot

Automated trading bot for Polymarket's 5-minute BTC Up/Down markets.

## Features

- 🎯 **Real-time prediction** based on BTC price movement (Binance API)
- 📈 **Confidence scoring** from 0-100%
- 💰 **Automated trading** with market or limit orders
- 📊 **SQLite database** logging of all trades and P&L
- 🖥️ **Web terminal** for live monitoring on port 10000
- 🧪 **Testnet support** for safe testing
- 🔒 **L2 authentication** via Polymarket Builder API

## Requirements

- Python 3.9+
- Polymarket account with Builder API credentials
- Polygon wallet with USDC (testnet USDC for testing)
